## Установка приложения
1. На данной стадии разработки для использования приложения необходимо установить `PyCharm`.
2. Клонируйте репозиторий с [GitHub](https://github.com/Dasha0Ramzer/online-store).
3. Установите зависимости:
```commandline
poetry install
```

## Разработка
Проект создан на основе фреймворка Django.

По состоянию на 08.06.2026 в проекте реализованы следующие приложения и их возможности:

- **Приложение `catalog`:**

1. Главная страница.
2. Контактная страница:
   - форма для обратной связи. После успешной отправки на странице появляется соответствующее сообщение.
